# VAANI-AI

A multilingual web application developed using HTML, CSS, and JavaScript.

This project allows users to translate text between multiple Indian and international languages through a simple and modern user interface.

Features:
✔ Translate text between languages  
✔ Support for Indian and global languages  
✔ Copy translated text  
✔ Swap languages  
✔ Responsive dashboard design  
✔ Beginner-friendly implementation  

Tech Stack:
• HTML  
• CSS  
• JavaScript  
• Translation API  

Future Improvements:
- Voice translation
- Dark mode
- Translation history
- AI-powered translation
